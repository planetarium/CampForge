# CampForge Usage Guide Glossary

| Term | Definition |
|------|-----------|
| **Camp** | 에이전트 온보딩 패키지 (Identity + Skills + Knowledge) |
| **domain-spec.yaml** | 캠프 생성의 입력 파일, 도메인 정의서 |
| **manifest.yaml** | 생성된 캠프의 메타데이터 파일 |
| **SKILL.md** | AgentSkills 호환 스킬 정의 파일 |
| **Adapter** | 플랫폼별 설치 스크립트 (claude-code, openclaw 등) |
| **Persona Level** | 에이전트 숙련도 (junior/mid/senior/lead) |
| **Scaffold** | campforge create가 생성하는 초기 캠프 뼈대 |
