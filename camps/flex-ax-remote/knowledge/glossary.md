# Flex AX Glossary

| Term | Definition |
|------|-----------|
| **인스턴스(instance)** | 결재 문서 한 건 |
| **템플릿(template)** | 결재 양식 |
| **결재선(approval_line)** | 문서의 승인/반려 단계 |
| **필드값(field_values)** | 결재 문서의 입력 필드 |
| **근태(attendance)** | 휴가/연차/반차 등 근태 기록 |
| **A2A** | Agent2Agent 프로토콜 |
| **a2x** | A2A 프로토콜 클라이언트 CLI |
| **SIWE** | Sign-In with Ethereum |
| **PostGraphile** | PostgreSQL 스키마에서 GraphQL API를 자동 생성하는 미들웨어 |
| **RLS** | Row-Level Security |
| **gq** | graphqurl CLI |
