# In-App Purchase Management Glossary

| Term | Definition |
|------|-----------|
| **Environment** | 실행 환경 구분 (production / staging / development) |
| **FungibleAsset** | 대체 가능한 게임 자산 (예: NCG, Crystal) |
| **FungibleItem** | 대체 가능한 게임 아이템 (소모품 등) |
| **ProductCategory** | IAP 상품 분류 체계 |
| **Store** | 앱스토어 구분 (0: Apple, 1: Google 등) |
| **CsvImportRequest** | CSV 기반 일괄 데이터 임포트 요청 포맷 |
| **Receipt** | IAP 구매 영수증. 스토어별 orderId로 추적 |
